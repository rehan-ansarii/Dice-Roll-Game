# Dice Roll Game (Python)

This is a simple dice roll game written in Python. In this game, the user plays against the computer. Both the user and the computer roll a dice, and the player with the higher number wins the round.

The game continues until either the user or the computer reaches 5 points.

## How the Game Works

1. The game asks the user to press Enter to roll the dice.
2. The user and the computer each get a random number between 1 and 6.
3. If the user's number is higher, the user wins the round.
4. If the computer's number is higher, the computer wins the round.
5. If both numbers are the same, the round is a draw.
6. Scores are updated after each round.
7. The first player to reach 5 points wins the game.

## How to Run

1. Clone this repository.

2. Open the project folder.

3. Run the Python file:

python dice_game.py

4. Follow the instructions in the terminal and press Enter to roll the dice.
